Événements de l'application Contact Tracer

CTEvents est un module qui contient les callbacks des événements de l'application. Il est possible d'ajouter des callbacks pour les événements suivants :

  • Changement des identifiants de l'appareil (onDeviceCredentialsChangedCallbacks)
  • Changement des identifiants du réseau (onNetworkCredentialsChangedCallbacks)
  • Changement de la configuration de surveillance (onMonitoringConfigChangedCallbacks)
  • Changement de l'état de santé (onHealthStatusChangedCallbacks)
